# PyPass

A lightweight cross-platform password manager using JSON and AES-128 encryption.

## JSON file structure
```json
{
    "items":[
        {
            "name":"Exmaple", "user":"Alice", "pass":"Prb1uDOZ3xWJN2C4wN/sKS0xS/Q2oJ6IjihY7UyHM2m0TmUMlUm9euLaw3CZ2qFD", "uri":"https://www.example.com/"
        }
    ]
}
```
